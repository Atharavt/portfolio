# 🌐 Personal Portfolio - Atharav Thakare

This is my personal portfolio website built to showcase my skills, projects, and ways to contact me.  
It's a fully responsive single-page site created using **HTML, CSS, and a little JavaScript**.

> 🔗 **Live Demo:** https://atharavthakareportfolio.netlify.app  
> 🧑‍💻 **GitHub Repo:** https://github.com/Atharavt/portfolio

---

## ✨ Features

- 🧑‍💻 **Hero Section** – Short introduction with a CTA to download my resume and contact me  
- 🛠️ **Skills Section** – Categorized skills with a “Currently Learning” strip  
- 📂 **Projects Section** – Featured projects with tech stack tags, Live Demo & Code links  
- 📬 **Contact Section** – Contact form powered by Formspree + social links  
- 📱 **Fully Responsive** – Looks good on mobile, tablet, and desktop  
- 🍔 **Mobile Navigation** – Clean hamburger menu on smaller devices  

---

## 🧰 Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (vanilla)**
- **Formspree** (for form submissions)
- **Netlify** (for deployment)
